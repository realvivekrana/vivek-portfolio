# Vivek's Portfolio

A modern, responsive portfolio website built with React.

## Features

- Smooth animations and transitions
- Fully responsive design
- Typing effect for role titles
- Interactive project cards
- Contact form with validation
- Social media integration
- Animated gradient effects

## Setup

1. Navigate to the project folder:
```
cd vivek-portfolio
```

2. Install dependencies:
```
npm install
```

3. **IMPORTANT - Add your profile photo:**
   - Save your profile photo as `vivek-profile.jpg`
   - Place it in the `vivek-portfolio/public` folder
   - The photo will automatically appear with a glowing border effect
   - If no photo is added, a beautiful gradient avatar with "VK" will show

4. Start the development server:
```
npm start
```

5. Build for production:
```
npm run build
```

## Customization

- Update personal details in component files
- Modify colors in CSS files (gradient colors: #667eea, #764ba2)
- Add more projects in `Projects.js`
- Add more skills in `Skills.js`

## Technologies Used

- React.js
- CSS3 (Animations & Gradients)
- Font Awesome Icons
- Responsive Design
- Glassmorphism Effects
